Logo Uses

Below are acceptable and unacceptable uses of the logo. The preferred version of the logo is full color on a white background. However, it is also acceptable to use the other versions shown below.

Full Color / Preferred

Black and White

Reverse Color

Reverse Color logo on bright image

Do

When using a brighter image with excessive white space, add a dark overlay to bring the one color logo forward.

Don't

The logo should always be clearly legible and should never recede into the photo.

Logo on colored backgrounds

Do

When pairing the logo with a colored background, avoid any visual vibration by pairing it with a reverse logo instead. If the background is a light color, use of the full color logo is permitted, provided it is legible.

Don't

The full color logo should never be paired with a background that will affect any part of the logo’s legibility, or cause any visual vibration.

Logo paired with text

Do
 

When using MaxPoint in a sentence or phrase, use "We are MaxPoint" rather than "We are [Logo]."

Don't

The logo should never be paired with text to read as a sentence or phrase.

Repeating Logos

The logo or brand mark should only be used one time on any page. Do not overuse it as a pattern, bullet points, or as any other decorative element.

Do
Don't

Clear Space

The MaxPoint logo should always have ample space between it and other visual elements. The minimum clear space around the logo should be the height of the ‘M’.

Minimum Logo Size

The minimum size requirements have been set to ensure legibility. Do not reduce the logo below the recommended size.
